`parse_html_tables`
===================

Convenience functions to render html tables into python data structures.

Uses the `html` module from the [`lxml`] package for processing of raw html text.

[`lxml`] is available from [pypi] or via `easy_install lxml`.

[`lxml`]: http://pypi.python.org/pypi/lxml/2.3alpha2
[pypi]:   http://pypi.python.org
